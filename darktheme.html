Variables and Constants

In JavaScript, you can define variables using var, let, or const. Here's a simple example:

const name = "John Doe";
let age = 30;
var city = "New York";
console.log(name, age, city);

Functions

Functions are reusable blocks of code in JavaScript. Here's an example of a simple function:

function greet(name) {
  return `Hello, ${name}!`;
}
console.log(greet("Alice"));
